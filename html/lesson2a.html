Here you will learn how to use the "internal" styling method as opposed to the "inline" method you used in lesson 1.
        For the internal style you declare a style tag within the head of you document and all the content in that tag is written as css.
        Some basic css selectors are
        tag
        .class
        #id
        Note the prefex before each selector. The italizized name is dependant on what you are tying to select.
        But the prefix is used to differentiate between
        an id a tag name and a class name
This is a div element with a class named 'classed'
The divs with the "classed" class use this css
.classed{
    background-color: #5bc0de;
    font-size: larger;
    }
This is a div element with an id of "iddd"
#iddd{
    text-align: center;
    border: dotted;
    border-bottom-color: orangered;
    background-color: khaki;

}
This too has been "classed
